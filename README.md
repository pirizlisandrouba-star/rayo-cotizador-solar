# ⚡ Rayo Express — Cotizador Solar Instantáneo

Cotizador online de energía solar fotovoltaica para **Rayo Soluciones Energéticas**.

## 🚀 Características

- **Lector de facturas con IA**: Subí tu factura de Edenor/Edesur y extraemos todos los datos automáticamente
- **Mínima fricción**: Solo 3 preguntas adicionales sobre tu techo
- **3 Paquetes estándar**: RAYO STARTER (4 paneles), RAYO HOGAR (8), RAYO TOTAL (12)
- **On Grid + Híbrido**: Toggle entre tipos de instalación
- **Triple Impacto**: Ahorro económico + ambiental + social
- **Precios configurables**: Editar `src/config/precios.json` sin tocar código
- **Mobile-first**: Diseñado para llegar desde Instagram/WhatsApp

## 📦 Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Deploy**: Vercel
- **IA (futuro)**: GPT-4o / Claude para lectura de facturas
